# Russia

Лендинг об интересных и захватывающих уголках России, представлены лучшие фото и видео

## Стек технологий

- css
- html
- webpack

## Команды

- Билд в режиме production
    ```
    npm run build
    ```
- Билд в режиме development
    ```
    npm run dev
    ```
- Запуск и отслеживание изменения файлов
    ```
    npm run start
    ```
